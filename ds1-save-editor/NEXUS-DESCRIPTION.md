# Dark Souls Remastered Save Editor - Portable Desktop Version

Portable desktop app for editing Dark Souls Remastered save files. Desktop version of [dsrsaveeditor.pages.dev](https://dsrsaveeditor.pages.dev/).

**⚠️ Work in Progress** - Active development, new features coming!

## Features

- **Portable** - Single .exe, no installation
- **Direct editing** - Overwrites save files directly
- **Auto-load** - Remembers your last save
- **Stats editing** - Level, VIT, ATN, END, STR, DEX, RES, INT, FTH
- **Resources** - Souls, Humanity, HP, Stamina
- **Safe Mode** - Auto-calculates Level/HP/Stamina
- **10 character slots**
- **Clean UI** - Dark theme, no menu bar
- **Offline** - Works completely offline

## Quick Start

1. Download `DS1 Save Editor 1.3.1.exe`
2. Run the .exe
3. Click "Find save in folder" → Select Documents folder
4. Select a character slot and edit
5. Click "Save" to overwrite original file

**Save location:** `%USERPROFILE%\Documents\NBGI\DARK SOULS REMASTERED\<user_id>\DRAKS0005.sl2`

## ⚠️ Important

- **ALWAYS BACKUP YOUR SAVES BEFORE EDITING**
- Close Dark Souls before editing
- Keep Safe Mode enabled (recommended)
- Edited saves may trigger anti-cheat online
